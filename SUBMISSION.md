THE FINAL CHATBOT IS multi_turn.py


Files:
1)   main.py - singleturn chatbot
2)   multi_turn.py - mutliturn chatbot


Features implemented:
1) Environment variables key handling
2) multiturn conversation memory
3) /reset resets the chatbot's history
4) /tokens give out the amount of tokens used
5) exit/quit commands
6) /number_chats command which gives number of prompts give by user
7) /history gives history of user prompts


model used:
openai/gpt-4o-mini
